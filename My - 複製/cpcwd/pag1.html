第一堂課：認識跨平台商業網站設計

瞭解網頁的組成、開發網頁的工具、網頁測試環境建置。

重點內容：
🔹HTML — 建立網頁骨架
🔹CSS — 控制樣式與排版
🔹JavaScript — 處理跟使用者之互動效果
🔹 PHP（伺服端語言）— PHP 是一種伺服端程式語言，用來製作動態網頁。 使用者開啟網站時，伺服器會處理PHP程式碼，再把結果回傳給用戶瀏覽器。 它是開源、跨平台，廣泛用於各類網站的後端開發。
🔹 MySQL（資料庫）— MySQL 是一套常見的資料庫管理系統（DBMS）， 用於整理、管理與儲存大量網站資料。 由於其開源、快速與穩定，廣泛應用於中小型與大型網站。
🔹 開發工具 — Visual Studio Code（VS Code）— 最常用的前端開發工具

第二堂課： HTML

HTML基本文件結構與CSS

重點內容：
🔹 HTML 文件結構 — <!DOCTYPE html>、<html>、<head>、<body> 等標籤的用途
🔹 常用 HTML 標籤 — 標題（<h1>~<h6>）、段落（<p>）、連結（<a>）、圖片（<img>）、清單（<ul>、<ol>、<li>）等
🔹 HTML 屬性 — id、class、src、href、alt 等屬性的使用方式
🔹 CSS 基本語法 — 選擇器、屬性與值的組成方式
🔹 內嵌 CSS 與外部 CSS 檔案的使用方法
🔹 常用 CSS 屬性 — 字型（font-family、font-size）、顏色（color、background-color）、邊框（border）、間距（margin、padding）等
🔹清除浮动

方法 優點 缺點 適用場景 作法
額外標籤法 相容所有瀏覽器 冗餘標籤 舊專案維護、極低版本瀏覽器相容 在浮動元素後加入空的區塊標籤，並設定 clear: both;
overflow法 程式碼簡潔 可能裁剪內容 簡單佈局、無溢出內容的場景 在父元素上設定 overflow: auto;或 overflow:hidden;
偽元素法 結構乾淨、無侵入 需寫 CSS 類別 絕大多數現代專案（首選） 在父元素的 CSS 中加入
    ::after {content: "";
      display: block;
      clear: both;}
父元素浮動 臨時解決 影響後續佈局 快速測試、臨時應急 在父元素上設定 float: left;或 float: right;
Flex/Grid 現代佈局、靈活 低版本瀏覽器需加前綴 新專案開發、響應式佈局 使用 display: flex;或 display: grid;於父元素

第四堂課：CSS

重點內容：a標籤應用與定位

🔹 a 連結
1.a :visited，點擊後
2.a :hover，鼠標覆蓋時
3.a :active，鼠標點擊時
4.a :link，未點擊前
5.a {text-decoration: none;}，去掉下劃線
🔹 定位方式
1.static，默認定位
2.relative，相對定位
3.absolute，絕對定位
4.fixed，固定定位
5.sticky，粘性定位
6.z-index，層級控制
7.position 屬性與 top、right、bottom、left 屬性的配合使用
8.定位的應用場景，如導航欄、彈出層等
9.定位與其他 CSS 屬性的互動，如浮動（float）、彈性盒（flexbox）等
